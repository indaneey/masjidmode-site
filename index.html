About Masjid Mode

Masjid Mode helps Muslims reduce digital distractions during prayer, dhikr, and spiritual activities. The app temporarily blocks selected applications and silences notifications to help users stay fully present in worship.

How It Works

  • Automatically silence notifications during prayer
  • Block distracting apps during worship sessions
  • User-controlled schedules and permissions
  • All processing happens locally on your device

Privacy First

Masjid Mode respects user privacy. The app does not read personal messages or collect sensitive user data. Restrictions are applied only based on user configuration using Apple's Screen Time APIs.
